# GLM-4.5-flash Console Client

Этот проект представляет собой консольного клиента для взаимодействия с моделью GLM-4.5-flash от ZAI. Программа позволяет пользователю вводить текстовые запросы и получать ответы от модели в режиме реального времени.

## Основные возможности

- Поддержка диалога с моделью GLM-4.5-flash.
- Возможность ввода сообщений пользователем.
- Вывод ответов модели.
- Поддержка выхода из программы с помощью команды `quit`.

## Требования

- Python 3.7 или выше
- Установленные пакеты:
  - `zai`
  - `python-dotenv`

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/glm-console-client.git
   cd glm-console-client
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` в корне проекта и добавьте API-ключ:
   ```env
   ZAI_API_KEY=your_api_key_here
   ```

## Использование

1. Запустите программу:
   ```bash
   python main.py
   ```

2. Введите свои сообщения. Для выхода введите `quit`.

## Пример использования

```
==================================================
Консольный клиент GLM-4.5-flash. Введите 'quit' для выхода.
==================================================

Вы: Привет, как дела?

AI: Привет! У меня всё отлично, спасибо. Как у тебя?

Вы: quit
До свидания!
```
